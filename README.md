# Memory Palace VR - 기억의 궁전

Meta Quest(오큘러스)용 VR 기억의 궁전 학습 앱입니다. WebXR과 Three.js를 사용하여 브라우저에서 바로 실행할 수 있습니다.

## 기능

- 🏛️ **3D 가상 궁전**: 아름다운 3D 환경에서 학습
- 📦 **객체 배치**: 공간의 특정 위치에 학습 객체 배치
- 🎨 **커스터마이징**: 색상, 모양, 텍스트 편집 가능
- 💾 **자동 저장**: 로컬 스토리지에 자동 저장
- 🎮 **직관적 조작**: Meta Quest 컨트롤러로 쉽게 조작
- 🚶 **자유로운 이동**: VR 공간 내에서 자유롭게 이동 및 회전

## 요구사항

- Meta Quest 1, 2, 3 또는 Pro
- WebXR을 지원하는 브라우저 (Meta Quest Browser 권장)

## 설치 및 실행

### 방법 1: 로컬 서버 실행

```bash
# Python 3가 설치되어 있는 경우
npm start
# 또는
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속

### 방법 2: GitHub Pages로 배포

1. GitHub 저장소 설정 → Pages에서 활성화
2. Meta Quest Browser에서 배포된 URL 접속

### 방법 3: 직접 파일 열기

Meta Quest Browser에서 `file:///path/to/index.html` 열기 (보안 제약으로 일부 기능 제한될 수 있음)

## 사용 방법

### VR 시작하기

1. Meta Quest를 착용합니다
2. Meta Quest Browser를 실행합니다
3. 앱 URL에 접속합니다
4. "VR 시작하기" 버튼을 클릭합니다

### 컨트롤러 조작법

#### 왼쪽 컨트롤러
- **조이스틱**: 앞/뒤/좌/우 이동
- **트리거**: 객체 선택 및 편집

#### 오른쪽 컨트롤러
- **조이스틱**: 좌/우 회전
- **트리거**: 새 객체 생성
- **B 버튼**: 선택한 객체 삭제
- **Y 버튼**: 수동 저장

### 학습 객체 관리

1. **객체 생성**: 오른쪽 트리거를 눌러 컨트롤러 앞에 객체 생성
2. **객체 선택**: 왼쪽 컨트롤러로 객체를 가리키고 트리거 클릭
3. **객체 편집**: 편집 패널에서 텍스트, 색상, 모양 변경
4. **객체 삭제**: 객체 선택 후 B 버튼
5. **저장**: Y 버튼으로 수동 저장 (자동 저장도 지원)

## 기억의 궁전 학습법

기억의 궁전(Memory Palace) 또는 로키 방법(Method of Loci)은 고대 그리스와 로마에서 사용된 기억 기법입니다.

### 사용 팁

1. **공간 활용**: 궁전의 각 위치에 기억할 내용을 배치
2. **시각화**: 각 객체를 독특하고 기억하기 쉬운 형태로 만들기
3. **경로 만들기**: 정해진 경로를 따라 객체들을 배치
4. **반복 학습**: 같은 경로를 여러 번 걸으며 복습
5. **연상 작용**: 객체의 위치, 색상, 모양과 내용을 연결

### 학습 예시

- **단어 암기**: 각 단어를 다른 색상/모양의 객체로 표현
- **순서 암기**: 궁전 입구부터 시계 방향으로 배치
- **개념 학습**: 복잡한 개념을 여러 객체로 분해하여 배치
- **시험 준비**: 시험 범위를 섹션별로 나누어 각 위치에 배치

## 기술 스택

- **Three.js**: 3D 그래픽 렌더링
- **WebXR**: VR 기능 지원
- **LocalStorage**: 데이터 저장
- **JavaScript ES6+**: 모던 JavaScript

## 문제 해결

### VR 모드가 작동하지 않는 경우
- HTTPS 연결을 사용하고 있는지 확인 (로컬은 http://localhost 가능)
- Meta Quest Browser를 최신 버전으로 업데이트
- 브라우저 설정에서 WebXR 권한 확인

### 객체가 저장되지 않는 경우
- 브라우저 캐시 및 쿠키 설정 확인
- 시크릿 모드가 아닌지 확인
- LocalStorage가 활성화되어 있는지 확인

### 성능 문제
- 너무 많은 객체(50개 이상)를 생성하지 않기
- 사용하지 않는 객체 삭제
- Quest를 재시작하여 메모리 정리
