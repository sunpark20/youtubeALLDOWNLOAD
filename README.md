# 핵심기능
YouTube 채널의 모든 동영상을 한번에 다운로드

## 주요 기능
- **채널 전체 다운로드** — 채널의 모든 영상을 한번에
- **재생목록 다운로드** — 단일 재생목록 또는 채널의 모든 재생목록
- **개별 다운로드** 단일 URL 입력으로 1개도 다운로드
- **중복 스킵** — 이미 받은 파일 자동 건너뛰기
- **화질 선택** — 음원만(m4a) / 360p / 720p / 1080p / 최고화질
- **Shorts 필터** — 채널 분석 시 3분 미만 영상 제외 옵션

## 만든 배경
제가 쓰려고 만들었어요.
1. 바다에서 근무합니다. 오프라인 환경에 한번에 다운받아가요.

  그래서 다운받아지면 260311_영상제목 으로 정렬되게 만들었어요.

## 한계
1. API키 추가 권장
2. 실리콘맥 우선 
  제가 실리콘맥을 쓰기 때문에, 인텔맥과 윈도우는 시간이 날때 업데이트 해줘야하는데, 이게 좀 어렵습니다. (가상환경에서 테스트해보기가 너무 힘듦)
3. 기능의 단순화
  기능을 이것저것 넣어봤더니 앱만 무거워지고 저에겐 별 쓸모가 없네요. 단순성을 우선하겠습니다.
  ex) 최고화질옵션로 빠니보틀 30분 영상 1편이 3GB ....
## 포맷
- **음원(m4a)**: AAC 원본 그대로 저장 (변환 없음, iPhone/Android 모두 재생 가능)
- **360p / 720p / 1080p**: H.264 우선, 없으면 다른 코덱으로 대체
- **최고화질**: 코덱 무관, 최고 해상도 다운로드 (4K VP9/AV1 등)

## 키보드 조작법
- URL 입력 → Enter → 화질 드롭다운 포커스
- 화질 변경 시 → 분석 버튼 포커스
- 화질에서 Enter (변경 없어도) → 바로 분석 시작
- 목록 가져오기 완료 → 받기 버튼에 포커스
- 받기 버튼에서 Tab → 영상 목록으로 포커스 이동
- 방향키(↑↓)로 목록 이동, Space로 선택/해제
- 목록에서 Enter → 다운로드 시작

## 기술 스택
- **Frontend**: HTML/CSS/JavaScript
- **Backend**: Python + FastAPI
- **Desktop**: pywebview
- **다운로드**: yt-dlp
- **빌드**: PyInstaller, Github Action
