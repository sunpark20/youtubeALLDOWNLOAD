# 🎬 YouTube Downloader Pro

YouTube 채널, 재생목록, 개별 영상을 쉽게 다운로드할 수 있는 macOS 앱입니다.

## ✨ 주요 기능

- ✅ **채널 전체 다운로드**: 채널의 모든 영상 한 번에
- ✅ **재생목록 다운로드**: 전체 재생목록 또는 개별 재생목록
- ✅ **개별 영상 다운로드**: 단일 영상 다운로드
- ✅ **쇼츠 필터링**: 3분 이하 영상 제외 가능
- ✅ **중복 방지**: 이미 다운로드한 영상 자동 건너뛰기
- ✅ **실시간 진행률**: 다운로드 속도, 남은 시간, 누적 용량 표시
- ✅ **자동 업데이트**: yt-dlp 엔진 자동 업데이트

## 🚀 빠른 시작

### 1. 앱 실행
다운로드한 `.app` 파일을 더블클릭하여 실행하세요.

### 2. 기본 설정
- **채널 핸들**: 다운로드할 YouTube 채널 핸들 입력 (예: `song_forest`)
- **저장 경로**: 폴더 브라우저로 다운로드 위치 선택
- **쇼츠 포함**: 체크 해제 시 3분 이하 영상 제외

### 3. 다운로드
원하는 탭을 선택하고 다운로드 버튼을 클릭하세요.

## ⚡ 빠른 체크 기능 (선택)

**대용량 채널을 자주 다운로드하시나요?**

YouTube API를 사용하면 다운로드 전 체크가 **10~100배 빠릅니다!**

### 설정 방법 (5분 소요, 완전 무료)

#### 1. Google Cloud Console에서 API 키 발급

1. [Google Cloud Console](https://console.cloud.google.com) 접속
2. 새 프로젝트 생성
   - 상단 "프로젝트 선택" → "새 프로젝트"
   - 프로젝트 이름 입력 (예: "YouTube Downloader")
3. "YouTube Data API v3" 활성화
   - 좌측 메뉴 "API 및 서비스" → "라이브러리"
   - 검색창에 "YouTube Data API v3" 입력
   - "YouTube Data API v3" 클릭 → "사용 설정"
4. API 키 생성
   - 좌측 메뉴 "API 및 서비스" → "사용자 인증 정보"
   - 상단 "+ 사용자 인증 정보 만들기" → "API 키"
   - 생성된 API 키 복사

#### 2. 앱에서 API 키 입력

1. 앱 실행
2. 좌측 사이드바 "⚡ 빠른 체크 (선택)" 확장
3. "🔑 YouTube API 키" 입력창에 API 키 붙여넣기
4. "🧪 API 키 테스트" 버튼으로 확인
5. "✅ 빠른 체크 활성화" 체크

### 무료 할당량

- **하루 10,000 units** (개인 사용 충분)
- 채널 체크 1회 ≈ 50~100 units
- 약 100개 채널 체크 가능/일

### API 키 없이도 사용 가능!

API 키가 없어도 앱은 정상 작동합니다. 단지 체크가 약간 느릴 뿐입니다.

## 📋 다운로드 모드

### 1. 전체 동영상 다운로드
채널의 모든 영상을 다운로드합니다.

**저장 위치**: `[채널명]/_모든 동영상/`

### 2. 전체 재생목록 동영상
채널의 모든 재생목록을 폴더별로 다운로드합니다.

**저장 위치**: `[채널명]/[재생목록명]/`

### 3. 개별 플레이 리스트
특정 재생목록 URL을 입력하여 다운로드합니다.

**입력 예시**: `https://www.youtube.com/playlist?list=PL...`
**저장 위치**: `플레이리스트/[업로더]_[재생목록명]/`

### 4. 단일 동영상
개별 영상 URL을 입력하여 다운로드합니다.

**입력 예시**: `https://www.youtube.com/watch?v=...`
**저장 위치**: `개별_영상/` 또는 `개별_음성/`

## 🎵 포맷 옵션

### 영상
- **MP4 (720p, 모바일용)**: 대부분의 기기에서 재생 가능
- **MKV (최고화질, 원본)**: 최고 화질 (용량 큼)

### 음성
- **MP3**: 음성만 추출 (최고 음질)

## 🔧 고급 기능

### yt-dlp 엔진 업데이트
다운로드가 안 될 때 사이드바 "🚀 엔진 업데이트" 버튼을 클릭하세요.

### 중복 다운로드 방지
이미 다운로드한 영상은 자동으로 건너뜁니다.
- 각 폴더마다 `.download_archive.txt` 파일로 관리
- 파일을 삭제한 경우 자동으로 아카이브에서 제거

## ⚠️ 주의사항

### API 키 관련
- ✅ **본인의 API 키만 사용하세요**
- ❌ 타인의 API 키를 사용하면 할당량 문제 발생
- ❌ API 키를 공개적으로 공유하지 마세요

### 저작권
- 개인적인 용도로만 사용하세요
- 저작권이 있는 콘텐츠 다운로드에 주의하세요

### 성능
- 대용량 채널(수백~수천 개 영상)은 시간이 오래 걸릴 수 있습니다
- 빠른 체크(YouTube API) 사용을 권장합니다

## 🛠️ 문제 해결

### 앱이 재실행되지 않아요
- 브라우저 탭을 닫고 다시 앱을 실행하세요
- 앱이 자동으로 기존 프로세스를 정리합니다

### 다운로드가 안 돼요
1. "🚀 엔진 업데이트" 버튼 클릭
2. 인터넷 연결 확인
3. YouTube URL이 올바른지 확인

### API 키가 작동하지 않아요
1. "🧪 API 키 테스트" 버튼으로 확인
2. Google Cloud Console에서 "YouTube Data API v3" 활성화 확인
3. API 키 제한 설정 확인 (IP, Referrer 제한 없음)
