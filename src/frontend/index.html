사용 안내

1. 채널 전체 다운로드 (@채널명)

YouTube 채널 페이지에서 @핸들을 복사해서 입력하세요.

입력 예시: @콤므

✓
재생목록별 폴더 구분 체크

@채널명 폴더 안에 재생목록별 하위 폴더를 만들어 다운로드합니다.

@채널명/재생목록A/영상1.mp4
✗
재생목록별 폴더 구분 체크 해제

모든 영상을 @채널명 폴더 하나에 다운로드합니다.

@채널명/영상1.mp4

2. 재생목록 다운로드

https://www.youtube.com/playlist?list=PLxxxxxxx

특정 재생목록의 모든 영상을 다운로드합니다.
