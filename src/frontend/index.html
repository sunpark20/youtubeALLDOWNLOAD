YouTube Data API 설정

API 키가 없어도 yt-dlp를 통해 채널 분석이 가능합니다. API 키를 설정하면 더 정확한 결과를 얻을 수 있습니다.

사용 안내

1. 채널 전체 다운로드 (@채널명)

YouTube 채널 페이지에서 @핸들을 복사해서 입력하세요.

입력 예시: @콤므

2. 재생목록 다운로드

전체 재생목록을 폴더별로 다운로드합니다.

입력 예시: https://www.youtube.com/playlist?list=PLxxxxxxx

3. 개별 플레이리스트 다운로드

특정 재생목록 내의 영상을 채널명/재생목록명 폴더에 다운로드합니다.

입력 예시: https://www.youtube.com/watch?v=xxx&list=PLxxxxxxx
